Boikhata MM is a cutting-edge, AI-driven educational ecosystem designed to empower students with personalized guidance, advanced study tools, and career development resources. From solving complex doubts to preparing for international standards like IELTS, Boikhata MM is your 24/7 academic companion.

🚀 **Explore the Live App:** [boikhata-ai-mentor.example.com](https://sohancreation.github.io/boikhatamm)

---

## ✨ Key Features

### 📚 Academic Excellence
- **AI Doubt Solver:** Instant, detailed explanations for complex academic queries across various subjects.
- **Smart Study Planner:** Custom-tailored schedules based on your goals, workload, and learning pace.
- **Subject Mentoring:** Deep dives into specific subjects with AI-driven insights.
- **Quizzes & Assessments:** Interactive testing to track progress and reinforce learning.

### 🌐 Global Education & IELTS
- **IELTS Master:** Comprehensive preparation for Speaking, Writing, Reading, and Listening modules with real-time feedback.
- **Study Abroad Consultant:** AI guidance on university selection, application processes, and visa requirements.
- **Scholarship Finder:** Personalized matches for international and local funding opportunities.

### 💼 Career & Professional Growth
- **AI Mock Interviews:** Realistic interview simulations with Vapi AI to build confidence and polish your skills.
- **Resume Builder:** Professional, ATS-friendly resume generation tailored to your industry.
- **Career Path Mapping:** Data-driven advice on choosing the right career based on your skills and interests.
- **Project Helper:** Guidance on academic and personal projects, from ideation to implementation.

### 🛠️ Productivity Tools
- **Slide Generator:** Create professional presentations from text prompts in seconds.
- **Smart Summarizer:** Condense long articles or textbooks into concise, digestible notes.
- **Research Mentor:** Specialized assistance for academic research and paper writing.

## 🛠️ Tech Stack

- **Frontend:** [React 18](https://reactjs.org/) with [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite 5](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Backend-as-a-Service:** [Supabase](https://supabase.com/) (Auth, Database, Edge Functions)
- **AI Integration:** [OpenAI](https://openai.com/), [OpenRouter](https://openrouter.ai/), and [Vapi AI](https://vapi.ai/)
- **Deployment:** [GitHub Pages](https://pages.github.com/) via GitHub Actions

## 🚀 Getting Started

### Prerequisites
- **Node.js:** v20 or higher
- **npm:** v10 or higher

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sohancreation/boikhatamm.git
   cd boikhatamm
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env` file in the root directory and add your credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_key
   ```

4. **Launch Development Server:**
   ```bash
   npm run dev
   ```
